About me

I am a disabled army veteran, a former 25B distinguished graduate. I have received numerous awards and recognition. I have worked full time as a full stack .Net developer for the past 4+ years. Before that I did part time development using ColdFusion as the server side code. I am capable with HTML, CSS, JavaScript, jQuery, BootStrap, SQL, C#, ColdFusion and PowerShell.

I have also worked as a help desk technician, systems administrator and system analysis. I can install OS’es, software and hardware. I have also worked with various virtualization technologies such as VMware, hyper-V and virtualbox. I have help successfully complete 3 DIACAP (now RMF) security evaluations with an authority to operate (ATO). I am able to troubleshoot various issues at all levels across software, hardware, network, virtual and firewalls.

I have utilized various tools and technologies to create ‘Morning Report’ for the hospital commander. It showed the total utilization of all appointments with statuses for each department. To build this I used ColdFusion, HTML, CSS, Javascript, SQL and SSIS. This report was previously done partially by hand. 2 personnel would come in early every day at 6am and work on it until 9am. After I published the application it now took them about 30 mins to complete the report and that was mostly being able to see the data in a timely manner and making corrections/adjustments. I received a bonus and a Commanders Award for Civilian Service medal for the work I performed.

I recently created a COVID-19 tracker. This was written in C#, ASP.NET, HTML, CSS, JavaScript and SQL. This allows the users to input patients, update as the contact tracers are able to reach each patient. It generates several reports including; showing the previous day stats and a daily report to senior management. It also allows them to mark patients as different types for the different reports as needed. They can also update the case and report numbers as needed to keep the sequence numbers aligned.

Coding samples

Some sample coding projects.

The One Hundred (TOH)

Live life one loop at a time. A javascript coding exercise using loops. Write a program that displays the numbers from 0 - 100. Display the even numbers in BOLD. Click to see it in action!

Rewind

There and back again. A javascript coding exercise using arrays. Write a program that displays a string in reverse.

Click to see it in action!

FizzBuzz

Write a short program that displays each number from 1 to 100. For each multiple of 3, display "Fizz" instead of the number. For each multiple of 5, display "Buzz" instead of the number. For numbers which are multiples of both 3 and 5, display "FizzBuzz" instead of the number.

Click to see it in action!

Taco Cat

Write a program to determine if the word or phrase entered is a palindrome. A palindrome is a word or phrase that reads the same backward as forward.

Click to see it in action!

Loan Shark

A mortgage loan calculator application that takes in three parameters:
1. The amount of money loaned in dollars (balance)
2. The time over which the loan will be repaid, in months (term)
3. The percentage rate at which interest will accrue on the loan (rate)
